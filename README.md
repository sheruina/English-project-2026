# English-project-2026
English project game upload. Based on the novel The Ocean at the End of the Lane.
About how to download:
go to "English-project-2026", don't click into any files. See the green "Code" button? Click it and choose to download as zip. Find the downloaded zip file, open it and select all the files in the folder (eng project test.html and images folder). Move them to another folder on desktop(I don't know why it works but if you directly open the html file in the zip folder, the images cannot show). Click the html file and you can play now. ;)
